# Road-Change-Detection-from-High-Resolution-Satellite-Imagery

**📖 Project Description**
This project presents a deep learning-based framework that combines Convolutional Neural Networks (CNNs) and Transformer architectures for accurate road network extraction and change detection from high-resolution satellite imagery. The model leverages CNNs for capturing local spatial features and Transformers for understanding global contextual relationships. It is designed to work on multi-temporal satellite images, enabling the detection of road changes over time.

**🎯 Objectives**
1.Extract road networks from satellite images
2.Detect changes in road infrastructure across time
3.Improve segmentation accuracy using hybrid architecture
4.Handle complex urban and rural scenarios

**🧠 Methodology**
1.Preprocessing of satellite imagery
2.CNN-based feature extraction (e.g., U-Net backbone)
3.Transformer encoder for global context learning
4.Fusion of spatial and temporal features
5.Segmentation and change detection output
